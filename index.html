Landing page workshop.

Apprendre à coder et designer sa propre
landing page en HTML et CSS.

1. Commençons par le setup

  1. 1 - Ouvrez Sublime Text. un fenêtre noir devrait s'afficher.
  2. 2 - Créez un dossier landing sur votre bureaux. Glissez & déposez ce dossier dans Sublime Text window.
  3. 3 - Dans la navigation gauche de Sublime Text :
    • Clique droit "New file"
    • Enregistrez sous en le nommant index.html avec Cmd + S ou Ctrl + S
    • Réitérez en créant le fichier style.css
    • Ensuite choisissez "New Folder" et créez un dossier images
  4. 4 - Pour finir double-cliquez sur index.html pour l'ouvrir dans Chrome

2. Ajoutons du contenu HTML

  • Pour gagner du temps, Faites clique-droit sur les images suivantes et sauvez les dans votre dossier images
  • Commencez ensuite à partir de ce code HTML dans index.html
<html>
  <head>
    <meta charset="utf-8">
    <title>Coding day landing page</title>
  </head>
  <body>
    <h1>Bienvenur sur Food Huggers</h1>
    <p>Économiseurs d'aliments réutilisables en silicone</p>
    <p>
      <a href="#">Contactez nous</a>
    </p>

    <img src="images/kiwi.svg" alt="icon kiwi" width="100">
    <h2>Préservez vos fruits</h2>
    <p>Et conservez toute leur fraîcheur</p>

    <img src="images/fridge.svg" alt="icon frigo" width="100">
    <h2>Un frigidaire en ordre</h2>
    <p>Fini les restes qui pourrissent au fond de notre frigo</p>

    <img src="images/earth.svg" alt="icon earth" width="100">
    <h2>Réduisez vos déchets</h2>
    <p>Les déchets alimentaires représentent 30% de nos déchets</p>

    <img src="images/bowl.svg" alt="icon bole" width="100">
    <h2>Dégustez sans modération</h2>
    <p>Vos 5 fruits et légumes par jour</p>

    <p>Landing page créée par ©Coding day</p>
  </body>
</html>

Extra - utiliser des icons personnels chiadés

  • Vous pouvez choisir vos icons sur IconStore, The Noun Project ou NucleoApp (ils ont des packs gratuits).
  • Tips: Ne manquez pas de surveiller les nouvelles ressources d'icons sur Product Hunt

3. CSS pour la font & couleur

  • Lier votre stylesheet style.css dans la section head de votre HTML

<head>
  <link href='style.css' rel='stylesheet'>
</head>

Copiez/collez ensuite le code CSS ci-dessous style.css et modifier le car il est intentionellement disgracieux

  • Pour les fonts utilisez Google Fonts
  • Pour les couleurs utilisez Chrome plugin Colorzilla pour récupérer les couleurs des autres sites
  • Vous trouverez aussi de belles inspirations couleurs sur Coolors ou Color Hunt
/* style.css */
body{
  background: rgb(245,245,245);
  color: green;
  font-size: 25px;
  font-family: Helvetica;
}
h1{
  font-family: Courier;
  color: red;
  font-weight: lighter;
  font-size: 10px;
}
h2 {
  font-family: Courier;
  color: pink;
  font-weight: lighter;
  font-size: 8px;
}
a {
  color: yellow;
}
a:hover {
  text-decoration: none;
  color: purple;
}

5. Nommer les tags

  • Nommer votre banner et footer en utilisant un id commme <div id="banner"> et <div id="footer">
  • Nommer vos fonctionnalités en utilisant une class comme <div class="card">
  • Ajoutez ensuite ce code CSS
#banner{
  text-align: center;
  background-image: linear-gradient(rgba(200, 200, 200, 0.3), rgba(200, 200, 200, 0.3)), url("https://source.unsplash.com/1600x900/?vegetables");
  background-size: cover;
  padding: 150px 0;
  color: white;
}
#banner h1 {
  color: white;
  font-size: 50px;
}
#banner p {
  color: white;
  opacity: 0.8;
  font-size: 30px;
  font-weight: lighter;
}
#footer{
  padding: 30px;
  background: rgb(30, 30, 30);
  color: lightgrey;
}
.card{
  padding: 50px;
  font-weight: lighter;
  text-align: center;
}

.container {
  width: 100%!important;
}

6. Fontawesome pour les icons

  • Ajoutez le stylesheet fontawesome dans votre head

<head>
  <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.5.0/css/all.css">
</head>

  • Ajoutez ensuite ceci dans le footer

<ul>
  <li><a href="#"><i class="fab fa-youtube"></i></a></li>
  <li><a href="#"><i class="fab fa-facebook"></i></a></li>
  <li><a href="#"><i class="fab fa-twitter"></i></a></li>
  <li><a href="#"><i class="fab fa-github"></i></a></li>
</ul>

  • Accompagné d'un peu de style CSS

#footer a {
  color: lightgrey;
  font-size: 15px;
}
#footer a:hover {
  color: white;
}

7. Bootstrap setup

  • Liez le stylesheet de Bootstrap dans le head section de votre HTML avant le lien de votre stylesheet

<head>
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css" integrity="sha384-BVYiiSIFeK1dGmJRAkycuHAHRg32OmUcww7on3RYdg4Va+PmSTsz/K68vbdEjh4u" crossorigin="anonymous">
  <link href='style.css' rel='stylesheet'>
</head>

  • Jouez avec class="btn btn-primary" et class="list-inline", vos premières classes Bootstrap
